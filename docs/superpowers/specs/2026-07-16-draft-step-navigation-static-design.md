# 拟稿助手五步导航取消悬浮设计

## 目标

拟稿助手顶部的五步导航保持在页面原始位置，随主内容正常滚动，不再悬浮或覆盖下方分析内容。

## 范围

- 仅调整 `DraftStepNavigation.vue` 的定位样式。
- 移除粘性定位、顶部偏移和为悬浮设置的层级。
- 保留五个步骤的顺序、状态、颜色、文案和无障碍属性。
- 保留移动端横向滚动行为。
- 不修改拟稿助手的数据、接口、步骤推进或侧栏布局。

## 实现

五步导航继续作为拟稿助手主滚动区域中的普通块级元素。导航本身的边框、背景、圆角和步骤状态样式不变，只取消 `position: sticky`、`top` 和 `z-index`，使其占据正常文档流空间并随内容滚动离开视口。

## 验证

- 定向测试检查五步导航不再使用 `sticky` 或 `fixed` 定位。
- 运行拟稿助手现有前端测试与 Vite 构建。
- 浏览器验证滚动时导航随页面移动，且不遮挡事件分析内容。
- 检查桌面端与移动端的步骤状态和横向滚动无回归。
